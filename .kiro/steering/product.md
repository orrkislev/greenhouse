# Product Overview

## תיכון החממה - Student Management Platform

A comprehensive Hebrew-language educational platform for "תיכון החממה" (The Greenhouse High School), focused on project-based learning, mentorship, and student development tracking.

## Core Modules
- **Project Management**: Student project creation, tracking, and completion
- **Study Pathways**: Academic field exploration and vocational guidance  
- **Research Tools**: Research project management and documentation
- **Mentorship System**: Staff-student mentoring relationships
- **Schedule Management**: Weekly scheduling and time management
- **Journal System**: Student reflection and progress logging
- **Admin Dashboard**: Staff oversight and student switching capabilities

## Key Features
- Project lifecycle management with task tracking
- Gantt charts and timeline visualization
- Google Calendar integration for scheduling
- Rich text editing and document management
- Staff impersonation system for student support
- Multi-database architecture (Supabase)
- Real-time collaboration and updates
- Mobile-responsive design with RTL support

## Target Users
- **Students**: Project management, learning tracking, reflection
- **Staff/Mentors**: Student oversight, guidance, administrative tasks
- **Administrators**: System management, user switching, analytics

## Design Philosophy
- Hebrew-first interface with comprehensive RTL support
- Clean, accessible design with stone/neutral color palette
- Mobile-responsive layout for all screen sizes
- Focus on student autonomy and self-directed learning
- Seamless staff-student interaction workflows
